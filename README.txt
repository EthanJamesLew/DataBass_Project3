Usage
  "add"               add song prompt
  "sartist <artist>"  search by artist
  "salbum <album>"    search by album
  "stitle <title>"    search by title
  "deltitle <title>"  delete an entry
  "delidx <idx>"      delete by index
  "ls"                list songs
  "save <filename>"   save current database to disk
  "load <filename>"   load database from disk
  "quit"              quit the program
